## 一、从"会跑任务"到"会接活"：4 个心法

新手把 Hermes 当一次性的脚本跑，老手把它当**一个 24 小时帮你（或客户）干活的运营机器人**。差别在这几点：

1. **任务拆成"步骤流"，别指望一句话搞定一切**。复杂活先让它「列个分步方案给我看」，确认后再让它跑。Hermes 的强项就是把"第一步 → 第二步 → 第三步"自动串起来，你说得越像流程，它跑得越稳。
2. **能定时就定时，让它自己上班**。获客播报、数据巡检这类活，配成**定时任务**让 Hermes 自己按点跑——这才是 agent 比脚本值钱的地方：人不盯着，活也在干。
3. **接外部触发器，做成"留言/获客机器人"**。把 Hermes 接到留言箱 / 群消息 / 表单上：有人来 → 自动按话术回复 + 记档 + 通知你。这就是能直接卖给商家的"自动化系统"雏形。
4. **每个流程先小步验证，再放它自动跑**。先用控制台手动派一遍看它走通，再设成自动；敏感动作（发消息、写数据）先让它跑"演练版"确认无误，再上真数据。改坏了好回滚。

> 💡 **省钱关键**：Hermes 每步都可能调模型，烧 token。技巧：把任务范围和步数说清楚（「只抓这 3 个链接」「最多 5 步」），别让它漫无目的地反复试探。

## 二、5 个实战场景（照着改就能用）

### 场景 1 · 自动化获客：内容播报机器人
最适合自媒体 / 做私域的人。让 Hermes 定时盯几个信息源，整理成一条能直接发的内容：

> 每天早上 8 点，抓这 3 个网站的最新文章标题和链接，挑出和「AI 工具」相关的，整理成一段 100 字的早报，发到我的留言箱（或存成文件等我转发）。

定时跑起来，你每天醒来就有一条现成的获客素材。**这是把"信息差"变成"日更内容"的最省力玩法。**

### 场景 2 · 微信 / 群留言机器人
> 监听我的留言入口，有人发消息进来就按这套话术自动回复：先问需求，再发对应的资料链接，并把这条线索记到 `线索.xlsx`，最后通知我有新客户。

客服话术 + 自动记档 + 实时通知，一套跑通就是能卖给商家的**自动获客/接待系统**。

### 场景 3 · 定时抓取 + 播报
> 每隔 2 小时巡检这一批商品/链接的价格（或库存、状态），有变化就整理成对比表，发我一条提醒；没变化就不打扰。

监控、比价、舆情、状态巡检——这类"盯着看"的活最适合交给 agent 自动跑，自己用或按月卖给客户都行。

### 场景 4 · 多步长流程编排
> 把这件事一条龙跑完：① 按关键词查资料 → ② 汇总成一篇初稿 → ③ 存档到指定文件夹并命名 → ④ 在表格里登记一行 → ⑤ 通知我完成。每步做完简单汇报。

Codex/Claude Code 擅长"写"，Hermes 擅长"把一长串步骤自动接起来跑完"。**复杂流程自动化是 Hermes 区别于其它工具的杀手锏。**

### 场景 5 · 接私活：给商家搭自动化系统
不止帮个人省事，更能**按项目收费给商家搭系统**：把上面几个场景组合成"获客 + 接待 + 记档 + 播报"的一整套自动化流程，部署到商家的电脑或服务器上。

- 在 U-King 里把 Hermes 接好虾盘云驱动，配好定时任务和触发器，交付一套能自己跑的系统。
- 更系统的编排教程见官方：**[Hermes 进阶编排（u-king.org/学院）](https://www.u-king.org/?from=hermes)**。

> 取舍：搭系统比代写脚本门槛高，但**单价高、能按月维护收费**，是 Hermes 最值钱的变现方向（见第四节）。

## 三、提示词 / 配置模板库（复制改需求即用）

| 目的 | 模板 |
|---|---|
| 让它先规划流程 | `先别动手，把「___」这件事拆成分步方案给我看，我确认后你再一步步执行。` |
| 定时播报 | `每天 ___ 点，抓「___」的最新内容，挑出和「___」相关的，整理成一段 ___ 字播报发我。` |
| 留言机器人 | `监听「___」入口，有消息进来就按这套话术回复：「___」，并把线索记到「___」，再通知我。` |
| 定时巡检 | `每隔 ___，检查「___」有没有变化，有变化整理成对比表提醒我，没变化别打扰。` |
| 多步编排 | `把这件事一条龙跑完：① ___ → ② ___ → ③ ___，每步做完简单汇报。` |
| 控制步数省钱 | `这个任务最多 ___ 步，只处理「___」，别漫无目的地反复试探。` |

> 想要按行业分类的完整模板包（电商客服、自媒体获客、私域接待、数据监控），扫码找我领。

## 四、变现路径：Hermes 怎么帮你赚钱

| 路径 | 怎么做 | 门槛 |
|---|---|---|
| **给商家搭自动化系统** | 把获客 / 接待 / 记档 / 播报组合成一套，部署到商家机器，**按项目收费 + 按月维护** | 中高，最值钱 |
| **代搭留言/获客机器人** | 帮做私域、开网店的人搭自动回复 + 记线索机器人 | 中 |
| **定时监控 / 播报服务** | 帮客户盯价格、库存、舆情、链接状态，按月订阅 | 中 |
| **流程自动化外包** | 闲鱼 / 猪八戒接「自动抓取、定时任务、流程编排」单 | 中 |
| **知识付费 / 教学** | 教别人用 Hermes 做自动化（就像这本手册），引流到社群变现 | 低，会用 + 会讲 |

**核心逻辑**：大多数商家知道"要做自动化"，却不知道**一个人就能用 Hermes 搭出来**。你赚的是**信息差 + 交付力**的钱——而且"搭系统按项目收费、再按月维护"比代写脚本的单价高得多、更持续。
